# Wu Wraps — Vinyl Vehicle Wraps (Renton / Seattle)

Marketing website for **Wu Wraps**, Mark Wu's vinyl vehicle wrap shop based in
the Renton Highlands, serving the greater Seattle and Puget Sound area for 18+
years.

The design is an **80's Marvel / Jim Lee–era comic book** aesthetic: bold
primary inks (red / blue / yellow), heavy black outlines, halftone dot
textures, hard offset panel shadows, speed lines, and starburst captions — built
to feel fun, loud, and confident while staying fast and SEO-friendly.

## Stack

- **Next.js 16** (App Router, Turbopack) + **React 19**
- **Tailwind CSS v4**
- **TypeScript**
- Self-hosted fonts via `next/font/local` (Bangers, Archivo Black, Barlow) — no
  external font requests at runtime

## Pages

- `/` — Home (hero, trust, services, process, gallery preview, testimonials, areas, CTA)
- `/services` — Full service lineup + process
- `/gallery` — Featured builds
- `/about` — Mark Wu's story / the new shop
- `/contact` — Quote form, shop info, FAQ

## SEO

- Per-page metadata, Open Graph, and Twitter cards (`lib/metadata.ts`)
- `LocalBusiness` / `AutoBodyShop` JSON-LD structured data in the root layout
- `sitemap.xml` and `robots.txt` generated by Next metadata routes
- Local keyword targeting (vinyl wrap Renton, car wrap Seattle, fleet graphics, etc.)

## Content

All copy and business data live in `lib/data.ts`. Replace the comic-style SVG
car illustrations (`components/ui/comic-car.tsx`) and the badge logo
(`components/ui/brand-logo.tsx`) with real install photos / the official Wu
Wraps logo when available.

## Develop

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build
npm run start    # serve production build
```

Set `NEXT_PUBLIC_SITE_URL` to the production domain so canonical URLs, the
sitemap, and structured data resolve correctly.
